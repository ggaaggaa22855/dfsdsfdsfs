Уровень 1: Структура HTML

Ты оказался в пустом HTML-документе. Создай базовую структуру страницы с заголовком и параграфом:

<!DOCTYPE html>
<html>
<head>
<title>Моя страница</title>
</head>
<body>
</body>
</html>
Объяснение: HTML (HyperText Markup Language) - это язык разметки, который определяет структуру веб-страницы. Теги <h1> - <h6> используются для заголовков, <p> для параграфов текста.
💡 Подсказка: Используй теги <h1>Мой заголовок</h1> для заголовка и <p>Мой текст</p> для параграфа
🎨

Уровень 2: Стили CSS

Твоя страница выглядит скучно! Добавь стили, чтобы сделать её красивой:

/* Сделай заголовок синим и выровняй по центру */
h1 {
}
/* Добавь отступы для параграфа */
p {
margin: 20px;
}
Объяснение: CSS (Cascading Style Sheets) - это язык стилей, который определяет внешний вид HTML-элементов. Свойство color задает цвет текста, а text-align управляет выравниванием.
⚡

Уровень 3: Магия JavaScript

Добавь интерактивности! Создай функцию, которая реагирует на клик по кнопке:

// Найди кнопку и добавь обработчик события
const button = document.getElementById('myButton');
button.addEventListener('click', function() {
});
Объяснение: JavaScript - это язык программирования, который добавляет интерактивность на веб-страницы. Событие 'click' срабатывает при клике на элемент. alert() показывает всплывающее окно, а textContent изменяет текст элемента.
💡 Подсказка: Попробуй добавить alert('Кнопка нажата!'); или изменить текст элемента с помощью document.getElementById('element').textContent = 'Новый текст';
🔗

Уровень 4: Комбинированная атака

Создай полноценный интерактивный элемент - счетчик, который увеличивается и уменьшается при нажатии кнопок:

<!-- HTML -->
<div class="counter">
<button id="decrease">-</button>
<span id="count">0</span>
<button id="increase">+</button>
</div>
<style>
/* CSS */
.counter {
display: flex;
gap: 10px;
align-items: center;
}
</style>
<script>
// JavaScript
let count = 0;
</script>
Объяснение: В реальных проектах HTML, CSS и JavaScript работают вместе. HTML создает структуру, CSS отвечает за внешний вид, а JavaScript добавляет интерактивность.
